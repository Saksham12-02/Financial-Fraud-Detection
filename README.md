# 💳 Financial Fraud Detection System

A Machine Learning and Power BI based Financial Fraud Detection System that predicts fraudulent transactions and provides interactive business intelligence dashboards.

---

# Project Overview

Financial fraud is one of the biggest challenges in banking and fintech industries.

This project combines

• Machine Learning
• Data Analytics
• Power BI
• Streamlit

to detect suspicious transactions and visualize fraud trends.

# Installation

Clone Repository

```
git clone https://github.com/yourusername/Financial-Fraud-Detection.git
```

Install Dependencies

```
pip install -r requirements.txt
```

Run Streamlit

```
streamlit run app.py
```
